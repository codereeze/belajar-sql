## RENAME

- #### Rename: Table

```bash
RENAME TABLE nama_tabel_lama TO nama_tabel_baru;
```

- #### Rename: Column

```bash
ALTER TABLE nama_tabel RENAME COLUMN nama_kolom_lama TO nama_kolom_baru;
```

- #### Rename: Index

```bash
ALTER TABLE nama_tabel RENAME INDEX nama_index_lama TO nama_index_baru;
```
