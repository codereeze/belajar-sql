# User dalam Database MySQL

Dalam MySQL, pengguna (user) adalah entitas yang digunakan untuk mengelola akses ke server MySQL. Setiap pengguna memiliki identitas yang unik dalam sistem, termasuk nama pengguna (username) dan kata sandi (password).
